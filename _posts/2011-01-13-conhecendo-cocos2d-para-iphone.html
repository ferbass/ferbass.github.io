--- layout: post title: Conhecendo cocos2d categories: - Objective-C tags: - cocos2d - iphone - Objective-C status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '208604988' --- A pouco tempo comecei a estudar cocos2d, para quem não conhece cocos2d é um framework para construção de jogos 2D, demonstrações e aplicações interativas. É baseado no design cocos2d Python ele usa os mesmos conceitos, mas em vez de usar Python usa Objective-C. O framework é bem completo e combinado com outros frameworks como por exemplo ChipMunk , Box2d e SpaceManager o que vai faltar é apenas imaginação para criar uma aplicação ou game de sucesso. Motivos para usar o cocos2d Fácil de usar: ele usa uma API familiar, e vem com vários exemplos Rápido: usa o OpenGL ES melhores práticas e estruturas de dados otimizadas Flexível: ffácil integração com bibliotecas de terceiros Grátis: é open source Comunidade: cocos2d tem uma grande comunidade AppStore aprovado: mais de 1500 jogos AppStore já a utilizam, inclusive muitos vendedores de melhores jogos. cocos2d 0.99.5 O grande destaque dessa versão concerteza é o suporte a Mac, Retina Display e PVR textures. Integração com X-Code A instalação do cocos2d cria um template de aplicações em seu X-Code, isso facilita muita coisa e também tras exemplos de aplicações quando você cria um nova projeto. O cocos2d tras uma serie de efeitos e animações prontas para você usar em seu projeto, suporte a acelerômetro, texturas, layers e muito mais. Para quem gostou e se interessa por desenvolvimento de games vale apena dar uma olhada e conhecer melhor o framework link http://www.cocos2d-iphone.org/
